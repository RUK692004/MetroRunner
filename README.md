# 🏃‍♂️ 3D Endless Runner Game (Unity + C#)

A 3D endless runner game developed using Unity and C#, featuring smooth player controls, procedural ground generation, obstacle spawning, and a complete game loop with scoring, audio, and UI systems.

---

## 🎮 Gameplay Overview

The player runs endlessly on a dynamically generated track while avoiding obstacles and collecting coins. The game progressively challenges the player through continuous obstacle spawning and increasing difficulty over time.

---

## ✨ Features

- 🏃 Smooth player movement controls
- 🧱 Infinite ground generation using repeating tile system
- 🛣️ Lane-based road structure
- 🚧 Dynamic obstacle spawning system
- 🪙 Coin collection mechanics
- 💥 Collision detection system
- 🧮 Score system with best score tracking (PlayerPrefs)
- 🎮 Game Over and Restart functionality
- 🏠 Start Menu interface
- 🔊 Audio feedback for:
  - Coin collection
  - Game over
  - UI interactions

---

## 🛠️ Technologies Used

- Unity Engine
- C#
- Unity Physics System
- Unity UI System (Canvas)
- PlayerPrefs (for saving best score)
- Object-Oriented Programming (OOP)

---

## 🧠 Key Systems Architecture

### 1. Player System
Handles player movement, input controls, and collision detection.

### 2. Ground System
Uses a repeating tile mechanism to create an infinite runner environment.

### 3. Obstacle System
Randomized obstacle spawning with increasing variation and difficulty.

### 4. Score System
Tracks in-game score and stores highest score using persistent storage.

### 5. UI System
Includes:
- Start Menu
- Game Over Panel
- Restart & Quit functionality
- Score display

### 6. Audio System
Manages sound effects for gameplay and UI interactions.

## 🚀 How to Play

1. Open the project in Unity
2. Load the **StartMenu Scene**
3. Click **Start**
4. Avoid obstacles and collect coins
5. Try to achieve the highest score
